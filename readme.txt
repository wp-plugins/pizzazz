== Description ==

An attractive and simple portfolio.

== Installation ==

* 1) Extract the zip file and just drop the contents in the `wp-content/plugins/` directory of your WordPress installation
* 2) Activate the plugin from Plugins page.
* 3) Edit a page that you want your portfolio to appear on and paste `[pizzazz]` into it.
* 4) Add your images to Pizzazz within your Wordpress admin area.
